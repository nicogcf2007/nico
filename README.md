# NRDev - Portafolio Personal

Portafolio de Nicolas Rivera, desarrollador full stack.

## Tecnologías

- **Astro 5** — Static Site Generator
- **React 18** — Componentes interactivos
- **TypeScript** — Tipado estático
- **Tailwind CSS 3** — Estilos utilitarios
- **Three.js** — Fondo 3D interactivo
- **Framer Motion** — Animaciones declarativas
- **GSAP + ScrollTrigger** — Animaciones de alto rendimiento
- **Lenis** — Smooth scrolling

## Scripts

| Comando | Descripción |
|---------|------------|
| `npm run dev` | Inicia servidor de desarrollo |
| `npm run build` | Build estático en `dist/` |
| `npm run preview` | Previsualiza el build |
| `npm run deploy` | Build + deploy a GitHub Pages |

## Deploy

El sitio se despliega automáticamente a GitHub Pages via GitHub Actions al hacer push a `main`.
